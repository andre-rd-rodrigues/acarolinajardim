# Carolina Jardim - Psicologia

Professional website for Carolina Jardim, Clinical Psychologist & Psychotherapist.

**Website:** [acarolinajardim.com](https://acarolinajardim.com)

## About

This is a static website showcasing the professional services of Carolina Jardim, a licensed psychologist (Professional License #24697) and psychotherapist in training. The website provides information about her services, professional background, and contact information.

## Technology Stack

### Core Libraries

- **jQuery**: 3.5.1
  - **Latest version**: 3.7.1 (as of 2024)
  - **Note**: Consider updating to latest version for security patches

- **jQuery Migrate**: 3.0.0
  - **Latest version**: 3.4.1 (as of 2024)
  - **Note**: Consider updating for better compatibility

- **Bootstrap**: 4.4.1
  - **Latest version**: 5.3.3 (as of 2024)
  - **Note**: Major version update available (v5), but requires migration

### Animation & Effects Libraries

- **AOS (Animate On Scroll)**: Used for scroll animations
- **Owl Carousel**: Used for carousel/slider functionality
- **GSAP (TweenMax)**: Used for advanced animations
- **ScrollMagic**: Used for scroll-based animations
- **Jarallax**: Used for parallax effects
- **jQuery Stellar**: Used for parallax scrolling
- **jQuery Waypoints**: Used for scroll-triggered events

### UI Libraries

- **jQuery Fancybox**: Used for lightbox/image galleries
- **Isotope**: Used for filtering and sorting layouts
- **ImagesLoaded**: Used for image loading detection
- **jQuery Lettering**: Used for text animation effects

### Third-Party Services

- **Typeform**: Embedded form for appointment booking
- **Insights.io**: Analytics tracking
- **Hotjar**: User behavior analytics

## Features

## Development

### Local Setup

1. Clone the repository
2. Open `index.html` in a web browser or use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js (http-server)
   npx http-server
   ```

### Customization

- **Styling**: Edit `css/style.css` for custom styles
- **JavaScript**: Edit `js/custom.js` for custom functionality
- **Content**: Edit `index.html` for content changes

## Library Updates

### Recommended Updates

Before updating libraries, test thoroughly as some may have breaking changes:

1. **jQuery 3.5.1 → 3.7.1**: Minor update, should be safe
2. **jQuery Migrate 3.0.0 → 3.4.1**: Compatibility improvements
3. **Bootstrap 4.4.1 → 5.3.3**: Major version - requires migration
   - Check [Bootstrap 5 migration guide](https://getbootstrap.com/docs/5.3/migration/)

### Checking for Updates

To check for the latest versions of libraries:

```bash
# For npm packages (if using package manager)
npm outdated

# Or check individual library websites:
# - jQuery: https://jquery.com/download/
# - Bootstrap: https://getbootstrap.com/
# - AOS: https://michalsnik.github.io/aos/
# - Owl Carousel: https://owlcarousel2.github.io/OwlCarousel2/
```
